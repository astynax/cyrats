# cyrats

"The CyberRats" game

## Developing

### Commands

- ``lein autotest`` runs tests for ``.clj``/``.cljc`` files and wathes for file change to rerun
- ``lein fw`` launches figwheel
- ``lein prod`` builds the production version of JS-library
